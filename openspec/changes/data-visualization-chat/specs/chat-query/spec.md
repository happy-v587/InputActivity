## ADDED Requirements

### Requirement: Natural Language to SQL Conversion
The system SHALL convert natural language queries into SQLite-compatible SQL queries using the configured LLM.

#### Scenario: Generate SQL from natural language
- **WHEN** user submits a natural language query about their activity data
- **THEN** the system sends the query with database schema context to the LLM
- **THEN** the LLM returns a valid SQLite SQL query
- **THEN** the system executes the SQL and returns results

#### Scenario: Chart rendering
- **WHEN** SQL query results are returned
- **THEN** the system renders the data as an appropriate chart type (bar, line)

### Requirement: SQL Execution
The system SHALL provide a safe, read-only SQL execution path against the local database.

#### Scenario: Execute chart query
- **WHEN** a SQL query is generated by the LLM
- **THEN** the system executes it in a read-only mode
- **THEN** results are returned as rows with column names

#### Scenario: Prevent destructive queries
- **WHEN** the generated SQL contains INSERT/UPDATE/DELETE/DROP/ALTER
- **THEN** the system rejects the query and shows an error
